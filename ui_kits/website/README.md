# Website UI kit — ajared.ca

Editorial marketing site — the **"field manual"** look. iA Writer Quattro S throughout, teal hairline grid, terracotta heat accent, ruler marginalia on the left.

## Files
- `index.html` — mounts the kit. Click-thru navigation between Home / Capabilities / Contact views.
- `Ruler.jsx` — the signature left-column ruler (0→600, major/minor ticks, terracotta on 500).
- `Header.jsx` — sticky top bar with ajaRed wordmark + coordinates + primary nav.
- `Hero.jsx` — oversized display headline + canvas moiré animation.
- `OfferCards.jsx` — 2×2 grid of numbered offers (Advisory / Build / Learn / AI Search), flip-to-teal on hover.
- `Capability.jsx` — 001/002/003/004 capability rows with deliverables ticklist.
- `ContactBlock.jsx` — dark-teal footer CTA with blinking status light.
- `Footer.jsx` — 2-col footer with terracotta top-rule.

## Faithfulness
Rebuilt from Figma `/Super-Simple-Brand-Guide` + the public `Ajared/website` XSLT templates. Copy is lifted directly. Icons are Unicode + the CSS "arrow" primitive. No emoji.
